The best way to manage your task is to ace them!

Todos is a task managing application that lets
you manage, organize and take charge of your
task. It is easy to use with a responsive and
attractive web interface. So go ahead and own
your task, own success!

Technology Stack

In building this application, we harness the following technologies

  • HTML/CSS/JavaScript: For designing an interactive user interface.
  • Python (Flask)/Jinja: For developing the server-side logic.
  • SQLAlchemy (SQLite): For managing the database.
  • Git/GitHub: For version control and collaboration.
  • Google Engine App:Google Engine App: For deployment
  • Cypress: For automated testing
  • GoogleMeet/WhatsApp: For seamless and fast communication
